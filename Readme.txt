INSTALLATION
____________

1)- OS Réquis: TOUTES LES DISTRIBUTIONS WINDOWS, TOUTES LES DISTRIBUTIONS GNU/LINUX ; 

2)- installer le répertoire "steganographie" dans le dossier "www" de votre application EasyPHP, ou WampServer,ou 
Xampp, ou ... ;

3)- L'application ne nécessite aucune autre configuration particulière (étant donné sa simplicité :) )

NAVIGATION
__________

1)- Au niveau de votre navigateur web (Internet explorer, Mozilla Firefox, Opéra, Google chrome, ...) entrer 
"localhost" ou encore l'adresse de bouclage "127.0.0.1", puis appuyez sur la touche "entrée";

2)- Au niveau de la page d'acceuil de votre application serveur (EasyPHP, Wamp, ...) cliquez sur "steganographie",
l'application devrait alors se lancer sans problème;

3)- Pour vos tests j'ai eu la gentillesse de mettre une image "terre.bmp" à la racine du projet, bien sûre toute
autre image au format bitmap que vous choisirez vous meme fera également l'affaire;

4)- Pour plus d'info se référer à l'article "Dissimuler un message secret dans une image grâce à la Stéganographie." 
présent sur mon blog.
